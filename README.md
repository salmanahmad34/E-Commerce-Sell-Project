# 📊 E-Commerce Sales Dashboard | Power BI

## 📌 Project Overview

This project is an interactive **E-Commerce Sales Dashboard** built using **Microsoft Power BI**. It provides valuable business insights by analyzing sales, profit, order quantity, payment methods, product categories, and state-wise performance through interactive visualizations.

---

## 📷 Dashboard Preview

![E-Commerce Dashboard](https://github.com/salmanahmad34/E-Commerce-Sell-Project/blob/main/Screenshot%202026-06-21%20170400.png?raw=true)

---

## 🎯 Objectives

- Analyze overall sales and profit performance.
- Track product quantity sold.
- Identify top-performing states and product categories.
- Understand customer payment preferences.
- Monitor monthly profit trends.

---

## 📈 Dashboard Features

- 📌 Total Sales Amount KPI
- 💰 Total Profit KPI
- 📦 Total Quantity KPI
- 📊 Profit by State
- 🛍 Profit by Category
- 📦 Profit by Sub-Category
- 💳 Orders by Payment Mode
- 📅 Monthly Profit Analysis
- 📈 Quantity by Sub-Category
- 📆 Interactive Date Filter

---

## 🛠 Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Visualization

---

## 📊 Key Insights

- COD is the most preferred payment method.
- Madhya Pradesh generated the highest profit.
- Printers contributed the highest profit among sub-categories.
- Sarees recorded the highest quantity sold.
- Interactive filters allow dynamic business analysis.

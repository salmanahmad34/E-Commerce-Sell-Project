# 📊 E-Commerce Sales Dashboard | Power BI

## 📌 Project Overview
This project is an interactive **E-Commerce Sales Dashboard** developed using **Microsoft Power BI**. The dashboard provides insights into sales performance, profit analysis, order trends, payment methods, and product category performance.

## 🎯 Objectives
- Analyze overall sales and profit.
- Monitor order quantity and revenue.
- Identify top-performing states and product categories.
- Understand customer payment preferences.
- Track monthly profit trends.

## 📈 Dashboard Features

- **KPI Cards**
  - Total Sales Amount
  - Total Profit
  - Total Quantity Sold

- **Profit Analysis**
  - Profit by State
  - Profit by Category
  - Profit by Sub-Category

- **Sales Analysis**
  - Orders by Payment Mode
  - Quantity by Sub-Category
  - Monthly Profit Trend

- **Interactive Filters**
  - Date Slicer

## 🛠 Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Visualization

---

## 📊 Key Insights

- COD is the most preferred payment method.
- Madhya Pradesh generated the highest profit among all states.
- Printers contributed the highest profit among sub-categories.
- Sarees recorded the highest quantity sold.
- The dashboard enables quick business performance monitoring through interactive visuals.
